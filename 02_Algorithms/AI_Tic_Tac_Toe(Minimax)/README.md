## ğŸ§© How It Works

1. The game board is a **3x3 grid**.  
2. Players take turns placing **X** or **O** on the board.  
3. After each move, the game checks for:
   - Winner (row, column, or diagonal)  
   - Tie (board full with no winner)  
4. If playing against AI:
   - AI calculates the **best move** using **Minimax**  
   - Human makes their move  
   - Board updates and displays after each turn  
5. The game continues until there is a winner or a tie.
